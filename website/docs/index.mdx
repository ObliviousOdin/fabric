## The shortest useful path

On macOS, Linux, or WSL2, install Fabric and prove one model route before adding
more providers or automation:

```bash
curl -fsSL https://raw.githubusercontent.com/ObliviousOdin/fabric/main/scripts/install.sh | bash
fabric model
fabric status --deep
```

Then run `fabric` and ask it to complete a small, observable task in a non-secret
workspace. A model appearing in a picker is not the same as a successful request;
the [Quickstart](/getting-started/quickstart) shows the complete verification loop.

## One profile, every surface

Your selected profile owns the model routes, credentials, memory, skills,
approvals, and sessions used across supported interfaces.

<div className="docs-hub-strip">

- **See and compose work:** [Desktop](/user-guide/desktop) or [web dashboard](/user-guide/features/web-dashboard)
- **Stay in the terminal:** [CLI](/user-guide/cli) or [TUI](/user-guide/tui)
- **Keep a conversation running:** [Messaging gateway](/user-guide/messaging/)
- **Connect applications and editors:** [API server](/user-guide/features/api-server), [ACP](/user-guide/features/acp), or [MCP](/user-guide/features/mcp)

</div>

:::tip Verify public builds
Install from this repository or from a release published on the
[`ObliviousOdin/fabric` release page](https://github.com/ObliviousOdin/fabric/releases).
Published desktop packages should include the expected checksum and platform signature.
:::
